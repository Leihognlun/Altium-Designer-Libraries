## GPIO使用(WiringPi)-A5R

  按钮 | Switch | LED |
  ---|---|---|
  B1|GPIO.0|GPIO.7|
  B2|GPIO.3|GPIO.2|
  B3|GPIO.11|GPIO.10|
  B4|GPIO.22|GPIO.21|
  B5|GPIO.23|GPIO.26|
  B6|GPIO.25|GPIO.27|

  通道切换 | |  
  ---|---|
  EN|GPIO.12
  A0|GPIO.13
  A1|GPIO.14
  A2|GPIO.6
